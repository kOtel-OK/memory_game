## HTML5 Game on Phaser Library 

Simply HTML5 game based on Phaser library.

### Technologies
- JavaScript (ES6 Modules, Classes)
- Phaser v3.55.2
- Web Font Loader 
- HTML5
- CSS

### How to play
- Find all card pairs before time runs out
